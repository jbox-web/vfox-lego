# vfox-lego

[![CI](https://github.com/jbox-web/vfox-lego/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-lego/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [lego](https://github.com/go-acme/lego)

## Install

```
mise plugin install lego https://github.com/jbox-web/vfox-lego.git
```
